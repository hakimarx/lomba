## ğŸ¯ Gambaran Umum

**Aplikasi Lomba (Musabaqah)** adalah sistem manajemen perlombaan berbasis PHP yang digunakan untuk:
- Mengelola event/kompetisi
- Mengelola peserta dan kafilah
- Input dan laporan penilaian
- Manajemen data Hafidz (penghafal Al-Quran)
- Multi-level user access control

### Tech Stack

| Komponen | Teknologi |
|----------|-----------|
| **Backend** | PHP 7.4+ (Native) |
| **Database** | MySQL/MariaDB |
| **Frontend** | HTML, CSS, JavaScript (Vanilla) |
| **Server** | Apache (XAMPP/MAMP) |
| **Library** | PhpSpreadsheet (Excel export) |
